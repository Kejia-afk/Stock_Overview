# 股票复盘大师 (Stock Review Master)

## 项目简介

股票复盘大师是一个专业的股票复盘分析工具，帮助投资者系统化记录、分析和总结每日股票交易情况，提升投资决策能力和持续改进投资策略。本工具采用专业现代型设计，适合金融领域专业应用，为投资者提供全面的数据分析和复盘功能。

## 核心价值

- **问题解决**：投资者缺乏系统化工具记录、分析和总结每日股票交易情况，导致无法有效积累经验和持续改进投资策略
- **目标用户**：个人投资者、股票交易分析师、投资顾问、基金经理

## 设计风格

专业现代型设计，适合金融领域专业应用的设计风格。采用简约、沉稳的视觉元素，强调数据可视化的清晰度和专业性。配色方案倾向于深色背景搭配高对比度元素，减少视觉疲劳，适合长时间专注分析。这种设计风格能传达可靠性和专业感，同时保持界面整洁，让用户能集中注意力在数据分析上。

## 设计准则

### 色彩方案
- **主色调**：深蓝色(#1A2C42)作为背景，传达专业感和稳定性
- **辅助色**：较浅的蓝色(#2D4A6D)用于区域分隔和次级元素
- **强调色**：绿色(#00C853)表示上涨，红色(#FF3D00)表示下跌
- **中性色**：浅灰(#E0E0E0)用于文本和界面元素

### 排版
- **标题**：Roboto 18-24px，加粗
- **正文**：Open Sans 14-16px，标准粗细
- **数据数字**：Roboto Mono 14-18px，便于数据对齐和阅读

### 视觉层次
- 使用卡片式设计分隔功能区域
- 关键数据和指标需高亮显示
- 通过空白空间创建呼吸感，避免视觉拥挤

## 设计规范

### UI细节
- **卡片组件**：圆角5px，轻微阴影效果(box-shadow: 0 2px 5px rgba(0,0,0,0.1))
- **按钮样式**：填充式主要按钮，线框式次要按钮
- **图表**：使用网格线提高可读性，工具提示显示详细数据
- **交互元素**：鼠标悬停时有轻微放大或颜色变化效果，提供即时反馈

### 响应式需求
- **桌面优先**：支持1920x1080和1366x768分辨率
- **平板适配**：关键分析功能保留，次要功能可折叠
- **手机适配**：聚焦于数据查看，复杂分析功能简化

## 功能架构

### 1. 用户控制中心
- 用户注册/登录系统
- 个性化设置（风格偏好、分析指标选择）
- 数据备份与恢复功能

### 2. 市场概览面板
- 大盘指数实时展示（上证、深证、创业板等）
- 近期热门板块展示（包含热门持续天数）
- 市场情绪量化指标（上涨家数、昨日连板表现、涨停家数和跌停家数）

### 3. 每日市场情绪分析
- 市场赚钱效应分析（简称、详细描述及代表性个股）
- 市场亏钱效应分析（简称、详细描述及代表性个股）
- 支持添加相关股票作为举例说明

### 4. 热门板块分析
- 板块阶段展示（观察、启动、爆发、分歧、回流）
- 板块核心个股表现（中军股、高度股、弹性股）
- 自动展示当日热门板块（最多5个，支持展开更多）

### 5. 个股分析工具集
- 个股K线图展示（支持多种时间周期）
- 成交量分析图表

### 6. 交易复盘功能区
- 交易记录录入（买入/卖出记录、原因记录）
- 交易结果分析（盈亏统计、持仓周期分析）
- 交易决策回顾（决策点标记、成功/失败原因记录）
- 图形化交易路径回放

### 7. 明日策略
- 观察股展示（赚钱效应观察股和亏钱效应观察股）
- 每只股票展示所属板块和板块内地位（中军、高度、弹性、低位补涨等）
- 策略分析（观察股表现及相应策略）
- 支持多种策略类型（如负反馈卖出、一字板加单后选择弹性票套利等）

### 8. 知识积累库
- 市场复盘日记（结构化模板）
- 个人交易心得记录
- 标签系统（便于归类和检索）
- 复盘经验搜索功能

### 9. 数据导出与报告
- 自定义报告生成（日报、周报、月报）
- 数据导出（CSV、PDF格式）
- 分析结果分享功能

## 数据策略

本应用属于数据密集型应用，需要采用综合数据策略：

### 1. 公共数据获取
- 股票市场基础数据（价格、成交量等）通过公共金融数据API获取
- 大盘指数和行业板块数据通过市场数据API实时更新

### 2. 用户数据管理
- 用户交易记录、复盘笔记等个人数据存储在本地
- 通过浏览器本地存储(localStorage)保存用户设置和历史分析
- 设计支持未来云同步功能的数据结构

### 3. 接口抽象设计
- 股票数据提供者接口设计，支持未来切换不同数据源
- 用户数据管理接口，为未来云存储做准备
- 分析引擎接口，支持扩展分析模型

## 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **数据可视化**：ECharts 5
- **本地存储**：IndexedDB、LocalStorage
- **状态管理**：React Context API
